This example demonstrates a simple JavaScript snippet to compute quarterly profit.

Financial Formula

\[ \text{Net Profit} = \text{Revenue} - \text{Expenses} \]

\[ ROI = \frac{\text{Net Gain}}{\text{Cost of Investment}} \]

Use these formulas when explaining profitability and ROI.
